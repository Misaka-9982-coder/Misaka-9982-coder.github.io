AcWing 2. 01иғҢеҢ…й—®йўҳ

зҠ¶жҖҒиЎЁзӨә

$f(i, j)$ иЎЁзӨәдҪ“з§Ҝ j зҡ„жғ…еҶөдёӢ иЈ…дёӢеүҚ i дёӘзү©е“Ғзҡ„жңҖдјҳи§Ј

зҠ¶жҖҒи®Ўз®—
  1. $f(i , j) = max(f(i - 1, j), f(i - 1, j - v(i)) + w(i))$
  2. $f(i - 1, j)$иЎЁзӨәдёҚйҖүз¬¬ i дёӘзү©е“Ғзҡ„жңҖеӨ§д»·еҖј
  3. $f(i - 1, j - v(i)) + w(i)$ иЎЁзӨәйҖүжӢ©дәҶз¬¬ i дёӘзү©е“Ғзҡ„жңҖеӨ§д»·еҖјпјҢз¬¬ i дёӘзү©е“ҒеҚ дәҶ$v(i)$зҡ„з©әй—ҙпјҢд»·еҖјдёә$w(i)$

дјҳеҢ–пјҡ

зҠ¶жҖҒиЎЁзӨә

$f(j)$ иЎЁзӨәе®№йҮҸ j дёӢзҡ„жңҖдјҳи§Ј
$backup(i)$ж•°з»„еңЁдёӢдёҖиҪ®иҝӯд»ЈејҖе§Ӣд№ӢеүҚе°ҶеҪ“еүҚзҡ„$f(i)$ ж•°з»„еӨҮд»Ҫиө·жқҘ

зҠ¶жҖҒи®Ўз®—

жҜҸж¬Ўи®Ўз®—дёӢдёҖиҪ®зҡ„зҠ¶жҖҒзҡ„ж—¶еҖҷпјҢйңҖиҰҒдҪҝз”ЁдёҠдёҖиҪ®иҝӯд»Јжӣҙж–°зҡ„з»“жһңпјҢдҪҝз”ЁдёҖдёӘж–°зҡ„ж•°з»„еӨҮд»ҪдёҠдёҖиҪ®зҡ„иҝӯд»Јз»“жһңеҸҜд»ҘдҪҝдёӢдёҖиҪ®зҡ„и®Ўз®—дёҚеҸ—иҮӘе·ұиҝҷдёҖиҪ®и®Ўз®—з»“жһңзҡ„еҪұе“Қ

жӯЈеәҸи®Ўз®—пјҡ
$f(j) = max(f(j), backup(j - v(i)) + w(i))$пјҢе…¶дёӯ j д»Һ0йҖ’еўһеҲ°иғҢеҢ…жңҖеӨ§е®№йҮҸ

жӣҙз®ҖжҙҒзҡ„жңүдҪҝз”ЁйҖҶеәҸиҝӯд»ЈиҝӣиЎҢи®Ўз®— пјҢ дёҚйңҖиҰҒдҪҝз”ЁеҸҰеӨ–зҡ„ж•°з»„еӨҮд»ҪдёҠдёҖиҪ®зҡ„зҠ¶жҖҒпјҡ
$f(j) = max(f(j), f(j - v(i)) + w(i))$, е…¶дёӯ $j д»ҺиғҢеҢ…жңҖеӨ§е®№йҮҸйҖ’еҮҸеҲ°v(i)$

AcWing 3. е®Ңе…ЁиғҢеҢ…й—®йўҳ

зҠ¶жҖҒиЎЁзӨә

$f(i, j)$ иЎЁзӨәе®№йҮҸ j зҡ„жқЎд»¶дёӢпјҢ еүҚ i д»¶зү©е“Ғзҡ„жңҖдјҳи§Ј

зҠ¶жҖҒи®Ўз®—

$f(i, j) = max(f(i - 1, j - k v) + k w), k = 0, 1, 2 вҖҰ(j >= k v)$
зӯүд»·еҸҳеҪў

  1. $f(i,j) = max(f(i - 1, j), f(i - 1, j - v) + w, f(i - 1, j - 2 v) + 2 w, f(i - 1, j - 3 v) + 3 w, вҖҰ)$
  2. $f(i,j - v) = max( f(i - 1, j - v), f(i - 1, j - 2 v) + w, f(i - 1, j - 3 v) + 2 w, вҖҰ)$
    еҫ—пјҡ
    $f(i, j) = max(f(i - 1, j), f(i, j - vi) + wi)$
    дјҳеҢ–
    $f(j) = max(f(j), f(j - vi)+ wi)$

AcWing 4. еӨҡйҮҚиғҢеҢ…й—®йўҳ

зҠ¶жҖҒи®Ўз®—

$f(i, j) = max(f(i - 1, j), f(i - 1, j - k v) + k w)$

дјҳеҢ–
$f(j) = max(f(j), f(j - k v) + k w)$

AcWing 5. еӨҡйҮҚиғҢеҢ…й—®йўҳ II

зҠ¶жҖҒи®Ўз®—

$f(i, j) = max(f(i - 1, j), f(i - 1, j - k v) + k w)$

дјҳеҢ–
$f(j) = max(f(j), f(j - k v) + k w)$

жҺҘзқҖдјҳеҢ–
еҲ©з”ЁдәҢиҝӣеҲ¶зҡ„жҖқжғіж–№жі•пјҢе°Ҷзү©е“ҒжҢүз…§$1, 2, 4, 8, вҖҰ$ зҡ„зӯүжҜ”еәҸеҲ—д№ҹе°ұжҳҜ$0001, 0010, 0100,1000$иҝҷж ·зҡ„дәҢиҝӣеҲ¶еәҸеҲ—еҜ№зү©е“ҒиҝӣиЎҢжү“еҢ…пјҢеӨҡеҮәжқҘзҡ„зү©е“ҒзӢ¬иҮӘжү“еҢ…жҲҗдёҖд»ҪгҖӮ
дҫӢеҰӮ пјҡ $11 = 1 + 2 + 4 + 4$,йӮЈд№ҲеҲ©з”Ё$1, 2, 4, 4$ иҝҷеҮ дёӘж•°еӯ—е°ұеҸҜд»ҘиЎЁзӨә$11$д»ҘеҶ…зҡ„жүҖжңүж•°еӯ—
дҫӢеҰӮпјҡ $10 = 2 + 4 + 4$, $9 = 1 + 4 + 4$, $8 = 4 + 4$,вҖҰе…¶д»–ж•°еӯ—д№ҹжҳҜдёҖж ·зҡ„йҒ“зҗҶ
иҝҷж ·еӯҗжҜҸдёӘжү“еҢ…е®Ңзҡ„зү©е“Ғз»„еҸӘиҰҒйҖүжӢ©жӢҝжҲ–иҖ…дёҚжӢҝпјҢе°ұиғҪжҠҠиҝҷдёӘ==еӨҡйҮҚиғҢеҢ…==й—®йўҳжҠҪиұЎжҲҗ==01иғҢеҢ…==й—®йўҳпјҢдҪҝеҫ—ж—¶й—ҙеӨҚжқӮеәҰйҷҚдҪҺ

AcWing 9. еҲҶз»„иғҢеҢ…й—®йўҳ

зҠ¶жҖҒиЎЁзӨә

$f(i, j)$ иЎЁзӨә дҪ“з§Ҝ j дёӢеүҚ i з»„зү©е“Ғдёӯзҡ„жңҖдјҳи§Ј

зҠ¶жҖҒи®Ўз®—

$f(i, j) = max(f(i - 1, j), f(i - 1, j -v(i, k)) + w(i, k)))$
зӯүд»·еҸҳеҪў
$f(j) = max(f(j), f(j - v(i, k) + w(i, k)))$

AcWing 898. ж•°еӯ—дёүи§’еҪў

еҸҜд»ҘиҮӘдёӢиҖҢдёҠйҖүжӢ©дёӢж–№дёӨдёӘж•°еӯ—зҡ„иҫғеӨ§еҖјеҠ дёҠиҮӘе·ұжң¬иә«зҡ„еҖјпјҢзӣҙеҲ°дёүи§’еҪўжңҖдёҠж–№дёәжүҖжұӮзҡ„еҖј
д№ҹеҸҜд»ҘиҮӘдёҠиҖҢдёӢиҝӣиЎҢж“ҚдҪң

зҠ¶жҖҒиЎЁзӨәпјҡ$f[i][j]$иЎЁзӨәиө°еҲ°еқҗж Үдёә$(i, j)$зҡ„зӮ№зҡ„жңҖе°Ҹи·Ҝеҫ„е’Ң
зҠ¶жҖҒиҪ¬з§»ж–№зЁӢпјҡ

иҮӘеә•еҗ‘дёҠ $f[i][j] = a[i][j] + min(f[i + 1][j], f[i + 1][j + 1])$ ,жңҖеә•еұӮзҡ„$f[i][j] = a[i][j]$,$res = f[0][0]$

иҮӘйЎ¶еҗ‘дёӢ$f[i][j] = a[i][j] + min(f[i - 1][j], f[i - 1][j - 1])$,еҲқе§ӢзҠ¶жҖҒ$f[0][0] = a[0][0]$,$res = min(f[n]_{0,1, вҖҰ, n})$

AcWing 895. жңҖй•ҝдёҠеҚҮеӯҗеәҸеҲ—

зҠ¶жҖҒиЎЁзӨә

$f[i]$ иЎЁзӨәд»Ҙ$a[i]$ з»“е°ҫжңҖй•ҝзҡ„дёҠеҚҮеӯҗеәҸеҲ—зҡ„й•ҝеәҰеҖј

зҠ¶жҖҒи®Ўз®—

$f[i] = max(f[i], f[j] + 1),if (a[j] < a[i])$
$res = max(f[i])$
дҫӢпјҡ

AcWing 896. жңҖй•ҝдёҠеҚҮеӯҗеәҸеҲ— II

иҝҷдёӘж Ҳ дёҚз”ЁдәҺи®°еҪ•жңҖз»Ҳзҡ„жңҖй•ҝеӯҗеәҸеҲ—пјҢиҖҢжҳҜ д»Ҙstk[i]з»“е°ҫзҡ„еӯҗдёІй•ҝеәҰжңҖй•ҝдёәi жҲ–иҖ…иҜҙ й•ҝеәҰдёәiзҡ„йҖ’еўһеӯҗдёІдёӯпјҢжң«е°ҫе…ғзҙ жңҖе°Ҹзҡ„жҳҜstk[i]гҖӮзҗҶи§ЈдәҶиҝҷдёӘй—®йўҳд»ҘеҗҺе°ұзҹҘйҒ“дёәд»Җд№Ҳж–°иҝӣжқҘзҡ„е…ғзҙ иҰҒдёҚе°ұеңЁжң«е°ҫеўһеҠ пјҢиҰҒдёҚе°ұжӣҝд»Јз¬¬дёҖдёӘеӨ§дәҺзӯүдәҺе®ғе…ғзҙ зҡ„дҪҚзҪ®гҖӮ
иҝҷйҮҢзҡ„ жӣҝжҚў е°ұи•ҙеҗ«дәҶдёҖдёӘиҙӘеҝғзҡ„жҖқжғіпјҢеҜ№дәҺеҗҢж ·й•ҝеәҰзҡ„еӯҗдёІпјҢжҲ‘д»¬еҪ“з„¶еёҢжңӣе®ғзҡ„жң«з«Ҝи¶Ҡе°Ҹи¶ҠеҘҪпјҢиҝҷж ·д»ҘеҗҺд№ҹжңүжӣҙеӨҡжңәдјҡжӢ“еұ•гҖӮ

AcWing 897. жңҖй•ҝе…¬е…ұеӯҗеәҸеҲ—

зҠ¶жҖҒиЎЁзӨә

$f[i][j]$иЎЁзӨәaзҡ„еүҚiдёӘеӯ—з¬ҰдёӯдёҺbзҡ„еүҚjдёӘеӯ—з¬ҰдёӯжңҖй•ҝзҡ„е…¬е…ұеӯҗеәҸеҲ—зҡ„й•ҝеәҰ

зҠ¶жҖҒи®Ўз®—
  1. $a[i] ~!= ~b[j]$ж—¶
    a[i] дёҚеңЁжңҖй•ҝе…¬е…ұеӯҗеәҸеҲ—дёӯпјҢ $f[i - 1][j]$
    b[j] дёҚеңЁжңҖй•ҝе…¬е…ұеӯҗеәҸеҲ—дёӯпјҢ $f[i][j - 1]$
  2. $a[i] ~==~ b[j]$ж—¶пјҢ$f[i - 1][j - 1] + 1$
зҠ¶жҖҒиҪ¬з§»
  1. $f[i][j] = max(f[i - 1][j], f[i][j - 1]);$
  2. $f[i][j] = max(f[i][j], f[i - 1][j - 1] + (a[i] == b[j]));$

AcWing 902. жңҖзҹӯзј–иҫ‘и·қзҰ»

зҠ¶жҖҒиЎЁзӨә

$f[i][j]$иЎЁзӨәе°Ҷеӯ—з¬ҰдёІa зҡ„еүҚiдёӘеӯ—з¬Ұ иҪ¬жҚўжҲҗеӯ—з¬ҰдёІbеүҚjдёӘеӯ—з¬Ұзҡ„жңҖе°‘ж“ҚдҪңж¬Ўж•°

зҠ¶жҖҒи®Ўз®—
  1. еҲ йҷӨж“ҚдҪң пјҡ еҲ еҺ»a[i] д№ӢеҗҺ, aзҡ„еүҚ i - 1дёӘеӯ—з¬Ұе’Ң b зҡ„еүҚjдёӘеӯ—з¬ҰзӣёеҗҢпјҢ$f[i -1][j] + 1$
  2. жҸ’е…Ҙж“ҚдҪңпјҡ жҸ’е…Ҙ b[j] д№ӢеүҚ, aзҡ„еүҚi дёӘеӯ—з¬Ұе’Ң b зҡ„еүҚ j - 1дёӘеӯ—з¬ҰзӣёеҗҢпјҢ $f[i][j -1] + 1$
  3. дҝ®ж”№ж“ҚдҪңпјҡ е°Ҷ a[i] дҝ®ж”№дёә b[j] еүҚпјҢa зҡ„еүҚ i - 1дёӘеӯ—з¬Ұ дёҺ bзҡ„еүҚ j - 1дёӘеӯ—з¬ҰзӣёеҗҢ
    еҰӮжһң a[i] == b[j], $f[i - 1][j - 1] + 0$пјҢ еҗҰеҲҷa[i] != b[j], $f[i - 1][j - 1] + 1$
зҠ¶жҖҒиҪ¬з§»ж–№зЁӢ
  1. $f[i][j] = min(f[i - 1][j], f[i][j - 1]) + 1;$
  2. $f[i][j] = min(f[i][j], f[i - 1][j - 1] + (a[i] != b[j]));$
еҲқе§ӢеҢ–
  1. $f[0][i] = i;$ aзҡ„й•ҝеәҰдёә0пјҢ еҸҳжҲҗbзҡ„еүҚiдёӘеӯ—з¬ҰйңҖиҰҒж·»еҠ iдёӘеӯ—з¬Ұ
  2. $f[i][0] = i;$ bзҡ„й•ҝеәҰдёә0пјҢ еҸҳжҲҗbзҡ„еүҚiдёӘеӯ—з¬ҰйңҖиҰҒеҲ йҷӨiдёӘеӯ—з¬Ұ

AcWing 282. зҹіеӯҗеҗҲе№¶

еӣ дёәеҸӘиғҪеӨҹи®©зӣёйӮ»зҡ„дёӨе ҶзҹіеӯҗиҝӣиЎҢеҗҲе№¶пјҢжүҖд»ҘдёҚжҳҜе“ҲеӨ«жӣјз®—жі•пјҢдёҚиғҪдҪҝз”ЁиҙӘеҝғгҖӮ

зҠ¶жҖҒиЎЁзӨә

$f[i][j]$иЎЁзӨәе°Ҷз¬¬iе ҶзҹіеӯҗеҲ°з¬¬jе ҶзҹіеӯҗеҗҲе№¶жҲҗдёҖе Ҷзҹіеӯҗзҡ„жңҖе°Ҹд»Јд»·

зҠ¶жҖҒи®Ўз®—

$f[i][j]=min(f[i][k] + f[k + 1][j] + s[j] - s[i - 1])$
$k вҲҲ[i,j - 1]$

иҫ№з•ҢжқЎд»¶

$f[i][i] = 0$пјҢ еҗҢдёҖе ҶзҹіеӯҗдёҚз”ЁеҗҲе№¶
еҗҲе№¶иҝҷдәӣзҹіеӯҗзҡ„д»Јд»·дёә$f[1][n]$

AcWing 900. ж•ҙж•°еҲ’еҲҶ

еҸҜд»Ҙе°ҶиҝҷдёӘй—®йўҳжҠҪиұЎжҲҗе®Ңе…ЁиғҢеҢ…й—®йўҳ
жҠҠж•°еӯ—1пјҢ2пјҢ3пјҢвҖҰпјҢnеҲҶеҲ«зңӢжҲҗnдёӘзү©дҪ“зҡ„дҪ“з§ҜпјҢжҜҸдёӘзү©дҪ“еҸҜд»ҘдёҚйҷҗж¬Ўж•°дҪҝз”ЁпјҢй—®жҒ°еҘҪиЈ…ж»ЎдҪ“з§Ҝдёәnзҡ„иғҢеҢ…е…ұжңүеҮ з§Қж–№жЎҲ

зҠ¶жҖҒиЎЁзӨә

$f(i, j)$ иЎЁзӨә еүҚ $i$ дёӘж•ҙж•°$(1, 2, 3, вҖҰ, i)$жҒ°еҘҪжӢјжҲҗж•ҙж•° $j$зҡ„ж–№жЎҲж•°йҮҸ

зҠ¶жҖҒи®Ўз®—

е°Ҷеҗ„дёӘйӣҶеҗҲдёӯйҖүжӢ©дәҶ0дёӘiпјҢ 1дёӘiпјҢ2дёӘiпјҢ...зҡ„йӣҶеҗҲж•°еҖјзҙҜеҠ иө·жқҘ

  1. $f(i, j) = f(i - 1, j) + f(i - 1, j - i) + f(i - 1, j - 2 i) + f(i - 1, j - 3 i) + вҖҰ$
  2. $f(i,j - i) = f(i - 1, j - i) + f(i - 1,j - 2 i) + f(i - 1, j - 3 i) + вҖҰ$

еҫ—пјҡ$f(i, j) = f(i, j - i) + f(i-1, j)$

иҫ№з•ҢжқЎд»¶

$f[1][1] = 1$

зӯүд»·еҸҳеҪў
$f(j) = f(j ) + f(j - i)$
иҫ№з•ҢжқЎд»¶
$f[0] = 1$,е®№йҮҸдёә0пјҢеүҚ i дёӘж•°е…ЁйғҪдёҚйҖүд№ҹжҳҜдёҖз§Қж–№жЎҲ

зӯүд»·еҸҳеҪў

AcWing 338. и®Ўж•°й—®йўҳ

жұӮ$[a, ~b]$еҢәй—ҙдёӯзҡ„ж•°еӯ—$kпјҲk вҲҲ[0, 9]пјү$еҮәзҺ°зҡ„ж¬Ўж•°пјҢ
еҸҜд»Ҙе°ҶиҝҷдёӘй—®йўҳиҪ¬жҚўжҲҗжұӮж•°еӯ—$k$еңЁеҢәй—ҙ$[0, a]$дёӯеҮәзҺ°зҡ„ж¬Ўж•°пјҢи®°дҪң$count(b)$пјҢ
е’ҢеңЁеҢәй—ҙ$[0, b]$ еҮәзҺ°зҡ„ж¬Ўж•°пјҢи®°дҪң$count(b)$пјҢ$count(b) - count(a)$е°ұжҳҜжүҖжұӮзҡ„ж•°еӯ—еҮәзҺ°ж¬Ўж•°
жҖқжғізұ»дјјдәҺеүҚзјҖе’Ң
дҫӢпјҡ
з»ҷе®ҡж•°еӯ— $n = abcdef$ жұӮж•°еӯ— x еңЁз¬¬еӣӣдҪҚеҮәзҺ°зҡ„ж¬Ўж•°
еҲҶзұ»и®Ёи®ә

  • 1 еүҚдёүдҪҚж•°зҡ„еӨ§е°ҸиҢғеӣҙеңЁеҢәй—ҙ$[0, abc - 1]$дёӯпјҢз¬¬еӣӣдҪҚж•°дёә$x$пјҢеҮәзҺ°зҡ„ж¬Ўж•°дёә$(abc - 1 + 1) \cdot 100$
  • 2 . еүҚдёүдҪҚж•°зӯүдәҺ$abc$ з¬¬еӣӣдҪҚж•°иҰҒдёә $x$, еҲҷеә”ж»Ўи¶ізҡ„жқЎд»¶дёә $d >= x$
    в‘  $d < x$ : еҗҺдёӨдҪҚпјҢз¬¬еӣӣдҪҚдёәxзҡ„ж•°еӯ—еӨ§дәҺ$abcdef$,ж— и§Ј
    в‘Ў $d == x$:з¬¬еӣӣдҪҚдёә$x$ зҡ„ж•°еӯ—еҗҺдёӨдҪҚж•°еӯ—еҸ–еҖјиҢғеӣҙдёә$[0, ef]$пјҢдёӘж•°дёә$ef + 1$
    в‘ў $d~ > ~x$: з¬¬еӣӣдҪҚдёә$x$ зҡ„ж•°еӯ—еҗҺдёӨдҪҚж•°еӯ—еҸ–еҖјиҢғеӣҙдёә$[0, 99]$пјҢ дёӘж•°дёә100

AcWing 291. и’ҷеҫ·йҮҢе®үзҡ„жўҰжғі

еҪ“жЁӘзқҖж‘Ҷж”ҫзҡ„е°Ҹж–№еқ—ж‘Ҷж”ҫе®ҢжҲҗд№ӢеҗҺпјҢеү©дёӢзҡ„дҪҚзҪ®е°ұеҸӘиғҪж‘Ҷж”ҫз«–зқҖзҡ„е°Ҹж–№еқ—дәҶпјҢжүҖд»ҘжЈӢзӣҳдёҠзҡ„е°Ҹж–№еқ—жҖ»зҡ„ж‘Ҷж”ҫж–№жЎҲж•°йҮҸзӯүдәҺжЁӘзқҖж‘Ҷж”ҫе°Ҹж–№еқ—зҡ„ж–№жЎҲж•°йҮҸ

зҠ¶жҖҒиЎЁзӨә

$f(i, j)$иЎЁзӨәеүҚ$i - 1$еҲ—е·Із»Ҹж‘ҶеҘҪпјҢдјёеҮәеҲ°з¬¬ $i$ еҲ—зҡ„зҠ¶жҖҒжҳҜ $j$ зҡ„жүҖжңүж–№жЎҲпјҢ е…¶дёӯ $j$ жҳҜдёҖдёӘдәҢиҝӣеҲ¶ж•°пјҢз”ЁжқҘиЎЁзӨәе“ӘдёҖиЎҢзҡ„е°Ҹж–№еқ—жҳҜжЁӘзқҖж”ҫзҡ„пјҢдҪҚж•°дёҺжЈӢзӣҳзҡ„иЎҢж•°дёҖиҮҙпјҢ1иЎЁзӨәе“ӘеҮ иЎҢе°Ҹж–№еқ—жҳҜжЁӘзқҖдјёеҮәжқҘзҡ„пјҢ0иЎЁзӨәе“ӘеҮ иЎҢдёҚжҳҜжЁӘзқҖдјёеҮәжқҘзҡ„гҖӮ

зҠ¶жҖҒи®Ўз®—

  1. зҠ¶жҖҒиҪ¬з§»
    з¬¬ $i$ еҲ—жҳҜеӣәе®ҡзҡ„пјҢйҰ–е…Ҳд»Һз¬¬ $i - 2$еҲ—иҪ¬з§»еҲ° $i - 1$ еҲ—ејҖе§ӢпјҢеҒҮи®ҫжӯӨж—¶зҡ„зҠ¶жҖҒдёә $k$пјҢе®ғеҜ№еә”зҡ„ж–№жЎҲж•°еә”иҜҘдёә $f(i -1, k)$пјҢеҚіеүҚ $i - 2$ еҲ—йғҪж‘Ҷе®ҢдәҶпјҢдё”д»Һз¬¬ $i - 2$ еҲ—дјёеҲ°з¬¬ $i - 1$ еҲ—зҡ„зҠ¶жҖҒдёә $k$ зҡ„жүҖжңүж–№жЎҲж•°гҖӮ
    k йңҖиҰҒж»Ўи¶ід»ҘдёӢеҮ зӮ№иҰҒжұӮ пјҡ
    1.1. $k &j == 0$пјҢеҚі $k$ е’Ң $j$ дёҚжҳҜеңЁеҗҢдёҖиЎҢжЁӘзқҖж‘Ҷж”ҫзҡ„е°Ҹж–№еқ—пјҢд»–д»¬д№Ӣй—ҙж‘Ҷж”ҫзҡ„е°Ҹж–№еқ—дёҚдјҡеҸ‘з”ҹеҶІзӘҒ
    1.2. жЁӘзқҖзҡ„е°Ҹж–№еқ—ж‘Ҷж”ҫе®Ңд№ӢеҗҺпјҢеү©дҪҷзҡ„з©әзқҖзҡ„дҪҚзҪ®жҳҜз•ҷзқҖз»ҷз«–зқҖж‘Ҷж”ҫзҡ„е°Ҹж–№еқ—зҡ„пјҢй•ҝеәҰеә”иҜҘдёәеҒ¶ж•°гҖӮ

  2. жңҖз»Ҳзӯ”жЎҲзҡ„иЎЁиҫҫејҸеә”иҜҘдёә$f[m][0]$пјҢиЎЁзӨәз¬¬$m - 1$еҲ—е·Із»Ҹж‘Ҷж”ҫе®ҢпјҢдё”з¬¬$m$еҲ—жІЎжңүдјёеҮәзҡ„е°Ҹж–№еқ—пјҢе…¶зҠ¶жҖҒиЎЁзӨәзҡ„еҖј$k = 0$

  3. зҠ¶жҖҒи®Ўз®—
    $f[i][j] += f[i - 1][k]$пјҢе…¶дёӯ $i вҲҲ[1, m], jвҲҲ[0, 1 << n], kвҲҲstate[j]$

  4. иҫ№з•ҢжқЎд»¶
    $f[0][0] = 1$

AcWing 91. жңҖзҹӯHamiltonи·Ҝеҫ„

з”ЁдәҢиҝӣеҲ¶еәҸеҲ—жқҘиЎЁзӨәжүҖжңүиө°иҝҮзҡ„зӮ№
дҫӢеҰӮиө°иҝҮзӮ№$0пјҢ1пјҢ2пјҢ4$еҲҷиЎЁзӨәдёә$10111$пјҢиө°иҝҮ$0пјҢ2пјҢ3$еҲҷиЎЁзӨәдёә$01101$

зҠ¶жҖҒиЎЁзӨә

$f(i, j)$ иЎЁзӨәжүҖжңүиө°еҲ°зӮ№ $j$ пјҢдё”жүҖжңүиө°иҝҮзҡ„зӮ№жҳҜ $i$ зҡ„жүҖжңүи·Ҝеҫ„й•ҝеәҰ

зҠ¶жҖҒи®Ўз®—

$f[i][j] = min(f[i][j], f[i - (1 << j)][k] + w[k][j])$

е…¶дёӯ $iвҲҲ[0, 1 << n], j вҲҲ [0, n -1], kвҲҲ[0, n - 1]$

иҫ№з•ҢжқЎд»¶ пјҡ$f[1][0] = 0;$
жңҖз»Ҳз»“жһңпјҡ$f[(1 << n) - 1][n - 1]$ пјҢиЎЁзӨә $(1 << n) - 1$ жҜҸдёӘзӮ№йғҪз»ҸиҝҮпјҢ $n - 1$ иЎЁзӨәеҲ°иҫҫз»ҲзӮ№

AcWing 285. жІЎжңүдёҠеҸёзҡ„иҲһдјҡ

зҠ¶жҖҒиЎЁзӨәпјҡ

$f[u][0]$:д»Ҙuдёәж №иҠӮзӮ№зҡ„еӯҗж ‘дёӯпјҢuдёҚеңЁеңәзҡ„еҝ«д№җеҖјжңҖеӨ§еҖј
$f[u][1]$:д»Ҙuдёәж №иҠӮзӮ№зҡ„еӯҗж ‘дёӯпјҢuеңЁеңәзҡ„еҝ«д№җеҖјжңҖеӨ§еҖј

зҠ¶жҖҒи®Ўз®—пјҡ
  1. uиҠӮзӮ№дёҚйҖүж—¶пјҢеӯҗиҠӮзӮ№йҖүжӢ©дҪҝеҝ«д№җеҖјжңҖеӨ§зҡ„жғ…еҶө
    $f[u][0] += max(f[j][0], f[j][1])$
  2. uиҠӮзӮ№йҖүжӢ©ж—¶пјҢеӯҗиҠӮзӮ№дёҖе®ҡдёҚиғҪйҖү
    $f[u][1] += (f[j][0])$
