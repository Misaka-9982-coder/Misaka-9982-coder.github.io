银行会计基础

会计要素

会计要素

会计核算的要素是会计所要核算和监督的内容，分为：

  1. 资产
  2. 负责
  3. 所有者权益
  4. 收入
  5. 支出
  6. 利润

1，2，3 反映企业在某一时间点上的财务状况

4，5，6 反映企业在一段时间内的经营成果

会计要素-资产

资产是指过去的交易、事项形成的并由企业拥有或控制的资源，该资源预期会给企业带来经济利益。

按照流动性分类：

  1. 流动资产
  2. 中长期贷款
  3. 长期投资
  4. 固定资产
  5. 无形资产等

包括现金、贵金属、存放同业、贷款、应收账款、股权投资；以及固定资产、无形资产、在建工程、库存物资等。

会计要素-负债

负债是指过去的交易、事项形成的现时义务，履行该义务预期会导致企业经济利益流出。

按流动性分类：

  1. 流动负债
  2. 长期负债

包括吸收存款、向央行接口、同业存放款、应交税费、应付利息；以及应付职工薪酬等

会计要素-所有者权益

所有者权益时所有者在企业资产中享有的经济利益。其金额为资产减去负债后的余额。

包括实收资本、资本公积、盈余公积、一般风险准备和未分配利润等

会计要素-收入

商业银行日常活动中形成的、与所有者投入资本无关的企业经济利益的增加。

商业银行的收入按大类可以分为：

  • 营业收入
  • 投资收益
  • 营业外收入

营业外收入主要有利息收入、金融机构往来利息收入、中间业务手续费收入等，营业外收入主要有固定资产盘盈、账户清理收入、政府专项奖励等

会计要素-支出

商业银行日常活动中发生的、与所有者利润分配无关的企业经济利益的减少

商业银行的支出按大类可以分为：

  • 营业支出

    主要有

    • 利息支出
    • 金融机构往来利息支出
    • 手续费支出
    • 所得税
    • 营业费用等

  • 营业外支出

    主要有

    • 固定资产盘亏
    • 捐赠
    • 罚款等

会计要素-利润

商业银行在一定的会计期间的经营成果，是收入与支出的差额。它是反映经营成果的最终要素，差额为正是企业的利润；差额为负是企业的亏损。

商业银行的利润可以分为：

  • 营业利润
  • 利润总额
  • 净利润

会计要素见的平衡关系

资产 = 负债 + 所有者权益

利润 = 收入 - 支出

会计恒等式： 资产 = 负债 + 所有者权益 + 收入 - 支出

会计核算原则

  1. 权责发生制

    • 凡是当期应当得到的收入或应当负担的支出，不论款项是否实际收付，都应该计入当期的收入和支出；

    • 凡是不属于当期的收入或和支出，即使款项已在当期收付，也不应该计入当期的收入和支出

    应用场景：

    • 应收利息计提、应付利息计提
    • 利息摊销等

  2. 谨慎性原则

    在处理不确定的经济业务时持谨慎原则，对可以预见的损失或费用应予以记录，不得多记资产或受益，也不得少记负债或支出

    应用场景：

    • 资产损失拨备

会计科目与账户

会计科目的分类

会计科目是对会计要素按其经济内容或者用途做更具体的分类，是设置账户、处理账务所必须遵循的规则和依据，也是确定会计报表项目的基础

  1. 按科目性质分类：
    • 资产类
    • 负债类
    • 所有者权益类
    • 资产负债共同类
    • 损益类
  2. 按科目与会计报表的关系分类：
    • 表内科目：反映商业银行会计要素实际增减变化的会计科目。
    • 表外科目：不反映商业银行会计要素实际增减变化的会计科目。可以细分为：
      • 或有事项类
      • 委托代理业务类
      • 登记备查类

会计科目-资产类

反映银行各项资产要素的会计科目，反映银行资金的占有和分布。

余额反映在借方

会计科目-负债类

反映银行各项负债要素的会计科目，反映银行资金取得和形成的渠道

余额反映在贷方

会计科目-所有者权益类

反映银行各项所有者权益要素的会计科目，反映所有者投入的资本金、利润、利润分配情况

余额反映在贷方

会计科目-资产负债共同类

反映银行各项资产和负债双重性的会计科目，是过渡性科目，主要用于核算银行发生的资金往来债权债务的轧差。

余额方向双向反映。为借表示债权，为贷表示债务

会计科目-损益表

反映银行各项收入和支出的会计科目，分为收入、支出两类

收入类：余额方向反映在贷方

支出类：余额方向反映在借方

会计科目-表外科目

或有事项类越饿方向同资产或负债；

登记备查类余额方向为贷方

账户

​ 账户是根据会计科目设置的，用来对会计科目所包含的经济内容进行详细的分类核算，反映会计要素内容具体增减变化情况和结果的专门方法。

​ 科目余额账户有密切的关系，科目提供综合资料，账户提供详细资料，账户必须建立在会计科目下。

​ 按其开户对象，分为内部账户和客户账户两类

  • 客户账户：银行在经营中对交易的客户和往来行设置的账户

  • 内部账户：根据银行本身经营管理需要而设置的用于银行内部资金核算的账户。

    例如：

    • 现金
    • 应交税金
    • 手续费收入
    • 利息收入账户
    • 工资支出账户

科目和账户的关系

核算种类是比科目更为详细的经济活动疯了，每一明细账户建立在“账户核算种类”之下，当科目调整时，仅需调整该对应关系，从而屏蔽会计核算的变化对业务的影响

科目： 核算种类 = 1: N

核算种类： 账户 = 1: N

记账方法

记账方法

记账方法在账簿中登记经济业务所用的方法，即以会计凭证为依据，运用一定的记账原理和规则，吧经济业务记录到账簿上。

记账方法按记账时涉及的是一个账户还是一个账户还是两个以上账户，分为单式记账法和复式记账法。

记账方法的分类

  1. 单式记账法：每一项经济业务的发生只在一个账户中进行登记的方法。商业银行会计中，主要用于表外登记备查类账务的核算。领入、增加时记贷方，领出、减少时记借方，余额在贷方。
  2. 复式记账法：每一项经济业务，都要以相等的金额同时在两个或者两个以上相互关联的账户中进行登记的记账方法。目前商业银行使用的是借贷记账法。

表内科目：借贷记账法

表外科目：单式记账法

借贷记账法

  • 记账符号：借、贷，不固定表示增加或减少，需要按分类来判定增加/减少
  • 理论基础：资产 = 负债 + 所有者权益 + 收入 - 支出
  • 遵循规则：有借必有贷，借贷必相等
  • 会计分录：指明经济业务应借、应贷的账户名称和金额的记录。 三个要素：记账符号、记账内容（账户）、记账金额

借贷记账法的好处：

  • 可以通过账户记录了解各项经济业务的来龙去脉，直至了解全部经济活动的过程和结果
  • 可以通过平衡记账的关系，检查会计记录是否准确无误。

如何编写会计分录

  • 分析经济业务的性质，确定起涉及的会计账户及所属科目
  • 确定会计科目的分类（资产、负债、权益等）
  • 确定会计账户内容的增减及其金额的大小
  • 根据账户的余额方向，确定哪个账户记借方，哪个账户记贷方
  • 写出会计分录

如何判定借贷

资产 + 支出 = 负债 + 所有者权益 + 收入
余额方向借 余额方向贷
增加记借方 增加记贷方
减少记贷方 减少记借方
账户所属科目 余额方向 借 贷
资产 借 增加 减少
负债 贷 减少 增加
所有者权益 贷 减少 增加
收入 贷 减少 增加
支出 借 增加 减少
资产负债共同类 双向 先判断属于债权还是债务 先判断属于债权还是债务

增加记余额方向的同方向；

减少记余额方向的反方向

试算平衡

有借必有贷，借贷必相等 $\rightarrow $ 发生额平衡

资产 = 负债 + 所有者权益 + 收入 - 支出 $\rightarrow $ 余额平衡

总体平衡

  1. 期初借方余额合计 = 期初贷方余额合计

  2. 本期借方发生额合计 = 本期贷方发生额合计

  3. 期末借方余额合计 = 期末贷方余额合计

单个科目计算平衡

  1. 期初借方余额 + 本期借方发生额 - 本期贷方发生额 = 本期借方余额
  2. 期初贷方余额 - 本期借方发生额 + 本期贷方发生额 = 本期贷方余额

财务组织

财务组织

财务组织，是指账簿设置、记账流程和核对方法的有机集合。分为明细核算和综合核算

  1. 明细核算
    1. 分户账
    2. 登记簿
    3. 余额表
  2. 综合核算
    1. 科目日结单
    2. 总账
    3. 日计表

明细核算

明细核算是以账户为基础的核算，反映每个账户的资金增减变化的详细情况

核算流程： 会计凭证 $\rightarrow $ 分户帐 / 登记簿 $\rightarrow $ 余额表

分户账 / 登记簿

  • 银行会计的明细分类账
  • 明细核算的主要形式
  • 登记簿： 辅助性账簿

余额表

  • 余额校验工具
  • 总分核对工具
  • 利息计算工具

综合核算

综合核算是以会计科目为基础的核算，反映各科目资金增减变化的总体情况。综合核算不能办理具体业务，而只为编制会计报表提供数字资料

核算流程 ： 会计凭证 $\rightarrow $ 科目日结单 $\rightarrow $ 总账 $\rightarrow $ 日记表

科目日结单

  • 登记总账的依据
  • 当日各科目借贷方发生额汇总记录
  • 当日各科目会计凭证张数汇总记录

总账

以科目为单位设立的，按照会计凭证汇总记录的总账

  • 各科目汇总记录
  • 日 / 月 / 季 / 年总账
  • 期初余额、当期借贷发生额、当前余额

日计表

日计表是反映单日业务活动和轧平当日全部账务的主要工具。从中也可以体现出试算平衡关系。

  • 轧账工具
  • 根据科目总账编制的会计报表
  • 试算平衡的工具

总分核对

按照明细账户对应的科目，总账与分户账或余额表进行核对

会计报表

会计报表

会计报表是会计核算工作的数字总结，据以反映资产、负债、所有者权益和损益状况，是分析业务活动和经营成果的重要依据。

  1. 表内报表
    • 日计表
    • 月计表
    • 损益表
    • 资产负债表
    • 利润表
    • 现金流量表
    • …
  2. 表外报表
    • 表外日计表
    • 表外月计表
    • 表外年计表
    • …

资产负债表

资产负债表是反映商业银行在某一特定日期的财务状况的报表，反映期末资产、负债和所有者权益的余额。通过资产负债表可以了解商业银行在某一日期资产、负债总额和结构；可以了解股东的权益情况

损益表

反映各分行的损益科目从年初到月末的累积余额，并且反映年初到当月的纯损或纯益。如果表中年初到当月累积收入大于支出，则产生纯益；反之，则产生纯损。

利润表

反映各损益项目从年初至期末的余额，一般按季编制。
