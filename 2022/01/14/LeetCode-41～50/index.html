41. 缺失的第一个正数

桶排序：

  1. 若数字不为INT_MIN，则将该数字自减一次
  2. 使用桶排序，将每个数字放到对应索引的位置上
  3. 最后遍历一遍数组，若nums[i] != i,则返回i + 1

时间复杂度为$O(n)$,不使用额外空间，空间复杂度为$O(1)$

43. 字符串相乘

模拟

  1. 使用两个vector数组A，B从后往前存放两个字符串数字的每一位
  2. 模拟乘法规则，使用第三个vector数组C存放两个数组对应位的乘积，C[i + j] = A[i] * B[j],遍历两个数组每一位
  3. 将C中每个位上的数字取余再进位，去除前导零
  4. 将C数组中的数字逐个转换成字符串拼接

45. 跳跃游戏 II

  1. 状态表示: $f(i)$表示走i步最远可以到达的距离
  2. 初始化：$f(0) = 0$ 表示走0最远距离0
  3. i的每一次移动都可以到达$[i+1, i+nums(i)]$这个区间中的任意位置，从1遍历到n的过程中，通过i第一次可以到达j时，j的最少步数为i的最少步数加1。
  4. 状态转移: $f(i)=f(j) + 1$,j为$j + nums[j] >= i$的第一个值

双指针解法

  1. 定义指针 cur代表当前位置和dis代表最远可以到达的位置
    1. 移动cur和dis,记录最远可达位置nxt
    2. 答案+1
    3. 更新dis为nxt
  2. dis大于n-1视为可达，返回答案

50. Pow(x, n)

比较典型的一道快速幂题目，可以使用位运算实现，也可以使用递归的做法实现，这里采用位运算的实现方式
```cpp
class Solution {
public:
double myPow(double x, int n) {
//long long类型方便计算
typedef long long LL; 

    //记录指数幂是否为负数                            
    bool is_minus = n < 0;
    //res 记录答案
    double res = 1;

    //位运算，时间复杂度为O（log n）
    for (LL k = abs(LL(n)); k; k >>= 1) {

        //如果k的二进制串表示最末尾的一位为1，则res乘以x
        if (k & 1) res *= x;

        //x倍增
        x *= x;
    }
    if (is_minus) res = 1 / res;
    return res;
}

};
