71. 简化路径

模拟

使用res记录当前路径，name表示两个/直接的文件名

  1. name为..时，回到上一个目录，将res中最后一个/往后的字符全部删除
  2. name为空或者多余的/时，不做处理
  3. 其余字符则将/ + name 添加至res末尾

72. 编辑距离

状态表示

$f[i][j]$表示将字符串a 的前i个字符 转换成字符串b前j个字符的最少操作次数

状态计算
  1. 删除操作 ： 删去a[i] 之后, a的前i - 1个字符和 b 的前j个字符相同，$f[i -1][j] + 1$
  2. 插入操作： 插入 b[j] 之前, a的前i个字符和 b 的前 j - 1个字符相同， $f[i][j -1] + 1$
  3. 修改操作： 将 a[i] 修改为 b[j] 前，a 的前i - 1个字符 与b的前 j - 1个字符相同
    如果 a[i] == b[j], $f[i - 1][j - 1] + 0$， 否则a[i] != b[j], $f[i - 1][j - 1] + 1$
状态转移方程
  1. $f[i][j] = min(f[i - 1][j], f[i][j - 1]) + 1;$
  2. $f[i][j] = min(f[i][j], f[i - 1][j - 1] + (a[i] != b[j]));$
    初始化
  3. $f[0][i] = i;$ a的长度为0， 变成b的前i个字符需要添加i个字符
  4. $f[i][0] = i;$ b的长度为0， 变成b的前i个字符需要删除i个字符

73. 矩阵置零

  1. 使用两个变量r0,c0分别记录二维矩阵第一行，第一列是否有 0
  2. 使用第一行的值分别记录该数组每一列是否有 0
  3. 使用第一列的值分别记录该数组每一行是否有 0
  4. 把所有含 0 的行列都置 0

74. 搜索二维矩阵

  1. 二分搜索目标值在哪一行
  2. 二分搜索目标值应在哪一列
  3. 比较矩阵对应位置的值和目标值是否相等

75. 颜色分类

  1. 使用i,j,k三个指针分别记录1,0,2三个值的位置
  2. i指针从前往后开始移动，k指针从后往前移动

76. 最小覆盖子串

使用哈希表记录字符出现次数

1. 加入`s[i]`，更新哈希表
1. 减去`s[j]`的冗余
1. 检查当前窗口是否符合`T`中所有字符，更新答案

77. 组合

深度搜索每一层的所有可能

78. 子集

思路同 77. 组合 ，枚举出从 0 到数组长度的所有组合即可

79. 单词搜索

  1. 枚举整个二维矩阵的字符，若为首字母则进入DFS
  2. 一一枚举周围几个字符是否为单词正确顺序上的字符，是的话就标记，不是则回溯

时间复杂度为$O(n^{2} * 3^{k})$

80. 删除有序数组中的重复项 II

  1. 数组有序上升，相同元素一定相邻
  2. 定义指针k，从前往后扫描，若nums[i] != nums[k - 1]，则将 nums[i]插入k指针之后，这样数组里面就能保存最多两个重复的值
