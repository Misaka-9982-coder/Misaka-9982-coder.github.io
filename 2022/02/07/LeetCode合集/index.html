1. 两数之和

暴力枚举方法很简单：两重循环枚举下标 i,j，然后判断 $nums[i]+nums[j]$ 是否等于 target。
时间复杂度：由于有两重循环，所以复杂度是 $O(n^2)$

使用哈希表，循环一遍nums数组，每次枚举一个数时做两件事

  1. 判断 $target−nums[i]$ 是否在哈希表中；
  2. 将 $nums[i]$ 插入哈希表中；

时间复杂度：由于只扫描一遍，且哈希表的插入和查询操作的复杂度是 $O(1)$，所以总时间复杂度是 $O(n)$.

15. 三数之和

排序 + 双指针

  1. 枚举每个数，表示该数的nums[i] 已确定， 通过双指针 l = i + 1 和 r = n - 1 往中间靠拢，直到找到nums[i] + nums[l] + nums[r] == 0所有符合条件的搭配

  2. 在找符合条件搭配的过程中，假设sum = nums[i] + nums[l] + nums[r]

若sum > 0，则r往左走，使sum变小

若sum < 0，则l往右走，使sum变大

若sum == 0，则表示找到了与nums[i]搭配的组合nums[l]和nums[r]，存到res中

  1. 判重处理

确定好nums[i]时，l 需要从i + 1开始

当nums[i] == nums[i - 1]，表示当前确定好的数与上一个一样，需要直接continue

当找符合条件搭配时，即sum == 0,需要对相同的nums[l]和nums[r]进行判重出来

56. 合并区间

思想方法类似于贪心

  1. 先给每个区间按左端点排序，并定义好st,ed两个指针

  2. 遍历每个区间，一共会有三种情况：

    1. 当前区间被上一个区间完全覆盖
    2. 当前区间的起点小于上一区间的右端点，将ed更新为当前区间右端点
    3. 当前区间的起点大于上一区间的右端点，将{st, ed}添加进答案中，更新st,ed为当前区间的两个端点

124. 二叉树中的最大路径和

样例

输入: [1,2,3]

   1
  / \
 2   3

输出: 6

输出: 42

思路
  1. dfs(root)返回的是从该节点往下走的最大值
  2. ans记录最大路径和
  3. 每一层DFS用left,right两个变量记录当前节点左右两个儿子的最大的单向路径和
    1. $left = max(0, dfs(u.left))$
    2. $right = max(0, dfs(u.right))$
  4. 更新ans,$ans = max(ans, u.val + left + right)$
  5. 每一层DFS返回以当前节点为祖宗节点的单向路径的最大和，即$u.val + max(left, right)$

146. LRU 缓存

  1. key-value对的增删改使用一个哈希表来实现
  2. 通过自定义一个结构体Node来存储key-value对，使这些Node节点连接起来形成一个双链表
  3. 定义L,R两个节点分别指向双链表的表头和末尾
  4. 如果缓存达到最大容量，则删除双链表的最右侧节点，即使R -> left,同时释放哈希表的内存
  5. 插入节点时，将新节点插入到双链表的表头，即L -> right
  6. 遇到get或put操作时，通过哈希表键值，找到该节点，将其出去，插入到双链表的表头，然后更新哈希表

236. 二叉树的最近公共祖先

dfs(root, p, q)：返回从root根结点往下找是否找到p点和q点的状态码state

state表示找到p点和q点的状态码:

  1. 11表示p点和q点都能找到
  2. 10表示只能找到p点
  3. 01表示只能找到q点
  4. 00表示两个点都找不到

当前点root找到p点和q点的状态主要依赖如下几种情况：

  1. dfs(root -> left, p, q)
  2. dfs(root -> right, p, q)
  3. 当前点为p点，root == p
  4. 当前点为q点，root == q

找到最早能同时找到p点和q点的结点，即最早出现state = 3的结点

使用两个哈希表分别记录父节点，节点深度，通过不断向上追溯父节点，找到最近公共祖先

  1. 使用深度优先搜索遍历一遍整颗树，记录下对应的父节点和深度
  2. 先把深度较深的节点追溯到和另一个节点的深度一致
  3. 两个节点同时向上追溯，直到两个节点相同为止

543. 二叉树的直径

思路

  1. dfs(root)返回的是从该节点往下走的最大值

  2. ans记录最大路径和

  3. 每一层DFS用left,right两个变量记录当前节点左右两个儿子的最大的单向路径和

    1. $left = dfs(u.left)$
    2. $right = dfs(u.right)$

  4. 更新ans,$ans = max(ans, left + right)$

    **注意：**两结点之间的路径长度是以它们之间边的数目表示。

  5. 每一层DFS返回以当前节点为祖宗节点的单向路径的最大和，即$max(left, right) + 1$
